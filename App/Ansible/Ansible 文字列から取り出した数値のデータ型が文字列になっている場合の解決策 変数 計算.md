## [【Ansible】文字列から取り出した数値のデータ型が文字列になっている場合の解決策](https://qiita.com/masa2223/items/064cf53f05b6ad5012dc)

int_test3.yml
```
---
- name: int TEST
  hosts: localhost
  gather_facts: no
  vars:
    ansible_python_interpreter: /usr/bin/python3
    test_str: "aaa 123 bbb"
  tasks:
    - name: fact
      set_fact:
        get_number: "{{ test_str.split()[1] }}"

    - name: debug
      debug:
        msg: "get_number : {{ get_number | int + 7 }}"
```

実行結果<br>
```
PLAY [int TEST] *****************************************************************************************************************************************************************************

TASK [fact] *********************************************************************************************************************************************************************************
ok: [localhost]

TASK [debug] ********************************************************************************************************************************************************************************
ok: [localhost] => {
    "msg": "get_number : 130"
}

PLAY RECAP **********************************************************************************************************************************************************************************
localhost                  : ok=2    changed=0    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0
```
