## インデント
`>>` : インデントを追加。<br>
`<<` : インデントを削除。<br>
複数行の場合<br>
ex 4行インデント<br>
<pre>
4>>
</pre>
もしくはヴィジュアル行選択で選択後`>`
<br>

## 削除
### 行範囲削除
削除したい範囲の最初の行で、**`mm`**<br>
↓<br>
削除したい範囲の最後の行まで移動して、**`d'm`**<br>
※2文字目は任意の文字(mでなくても良い)<br>

### 行削除
:行番号 d : 特定の行を削除<br>
ex 10行目を削除<br>
<pre>
:10d
</pre>
<br>

## ヴィジュアル選択
`viw` : カーソル上の単語を選択<br>
`vi"` : ダブルクォーテーションの内部を選択<br>
`vit` : HTMLタグの内部を選択<br>
<br>

## スクロール
### 1行スクロール
`CTRL-E` : 1行下へスクロール<br>
`CTRL-Y` : 1行上へスクロール<br>
