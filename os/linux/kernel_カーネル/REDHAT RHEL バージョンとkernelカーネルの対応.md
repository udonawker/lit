## [Red Hat Enterprise Linux のリリース日](https://access.redhat.com/ja/articles/16476)

Red Hat Enterprise Linux リリースを検索するには、以下を実行します。<br>
```
$ cat /etc/redhat-release
```

カーネルバージョンを検索するには、以下を実行します。<br>
```
$ uname -a
```
